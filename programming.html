Programming
Computer programming and development are critical to the past, present, and future of geospatial systems and techniques. The increasing ubiquity and diversity of online, mobile, and desktop GIS platforms along with the inclusion of cyber-infrastructure components within the bounds of geographic information systems (e.g., supercomputing, wireless sensor networks) means that GIS researchers and professionals need to be fluent in multiple forms of programming, and the life-cycles of system and software development.

Streamlining Wildfire Ignitions Data Processing
for National Park Service
​Wildfire Management 

Description
Geospatial datasets sometimes require preprocessing before they can be used for analysis in a GIS. This project serves as an example case. Raw data from the National Park Service indicating points of fire ignition in parks in the Eastern US is modified and processed. The aims of which are to produce analysis-ready geospatial data products, in addition to the means to streamline the workflow through automation. Future fire ignition datasets can be assessed and modified for usability via a customized, user-friendly tool/application developed through ArcPy scripting.
Analysis Procedures
This project takes raw csv text file data and utilizes Python file objects to read and write text and HTML files, as well as generate and populate Python dictionaries for key-value item access for backend data modification/deletion tasks on data/attribute fields. Arcpy geoprocessing tools for Table to Table Conversion (csv txt file -> Arc dbf file), XY Table to Point feature layer mapping (Arc dbf -> Arc shp), Select Layer By Attributes for unique value records selection and feature layer creation through looping, and Arcpy Mapping module ArcGISProject class objects to access project maps and map layouts for deliverables.

Example Project Title

Description
Imperative to any Geospatial Analyst's skillset is the ability to understand the diversity of geospatial data sources and the various attributes/metadata that come with each. In this assignment, the goal is to explore and understand the basic properties of the provided geospatial data so that general observations and conclusions can be made based on a broad range of specific technical information. The importance of Coordinate Reference System (CRS) standards, reprojection between CRSs, and the basics of transforming acquired data into georeferenced, discrete representations of landscape features is emphasized in this assignment project.

Results
Similar methods of Graphical User Interface (GUI) menu options and Command Line Input (CLI) Console inputs were used for data management tasks such as copying and removing data, and importing and exporting Shapefiles, GeoTIFF rasters, and ASCII grid rasters. Additional data exploration – finding the working CRS, spatial extent, and resolution – was achieved through both methods, with the resulting information printing to the Console.

​
​Understanding the concept of georeferenced data, explaining differences between the basic coordinate reference systems (CRS), using CRS standards to find parameters for a given CRS, and understanding concepts of re-projection between CRSs. Results comparisons expanded on these facets amongst differing GIS systems.
Through the exploration of the provided datasets, this assignment achieved its objectives to introduce the use of GRASS GIS and ArcGIS Pro for geospatial data analysis purposes. Both the GUI – in both programs – and CLI interface – in GRASS – were explored and utilized to perform such tasks as setting the computational region, displaying and inspecting raster data in both 2D and 3D, displaying vector data, querying map data to view values or attributes of a location(s), and managing geospatial data.

​

Reflection
This project was foundational and key to developing competence - and confidence - in navigating the complex, nuanced realm of geospatial data sources and acquisition. In the nontrivial examinations of geospatial datasets between two differing GIS software programs, it is clear how much significant knowledge and hands-on experience this assignment provided﻿.
